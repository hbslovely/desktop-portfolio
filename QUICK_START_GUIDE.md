# Quick Start Guide - Text Editor Feature

## 🚀 Getting Started

Your Desktop Portfolio now has a fully-featured **Rich Text Editor**! Here's how to use it:

---

## 📖 Basic Usage

### Opening the Editor

**Method 1: Desktop Icon**
1. Find "Text Editor" icon on desktop (4th column, top)
2. Double-click to open

**Method 2: Start Menu**
1. Click Start button (bottom-left)
2. Go to "Productivity" category
3. Click "Text Editor"

**Method 3: Search**
1. Click search icon in taskbar
2. Type "editor" or "text"
3. Click on "Text Editor" in results

**Method 4: Edit Existing File**
1. Open Explorer
2. Navigate to any text file (.txt, .md, .html, etc.)
3. Right-click on the file
4. Select "Edit"
5. File opens in Text Editor

---

## ✍️ Creating Your First Document

1. **Open the Editor**
   - Use any method above

2. **Start Typing**
   - Click in the white editor area
   - Type your content

3. **Apply Formatting**
   ```
   Bold:      Select text → Click B
   Italic:    Select text → Click I
   Underline: Select text → Click U
   ```

4. **Add a Heading**
   - Type your heading text
   - Select it
   - Click H1, H2, or H3

5. **Create a List**
   - Click the bullet (•) or number (#) button
   - Type your list items
   - Press Enter for new items

6. **Save Your Work**
   - Click the 💾 Save button
   - Choose a folder (Documents, Pictures, Downloads)
   - Enter a filename
   - Click Save
   - File downloads to your computer!

## 💾 Saving & Exporting

### Save Dialog
When you click Save:
1. **Navigation bar** shows current location
2. **Folder list** shows available folders
3. **Double-click folders** to navigate into them
4. **Click breadcrumbs** to go back
5. **Enter filename** in the input field
6. **Click Save** to download

### Export Options
Located in the bottom status bar:
- **Export TXT** - Plain text without formatting
- **Export HTML** - Formatted document with styles

Both options download the file immediately.

## 🔧 Tips & Tricks

### Keyboard Shortcuts
- **Ctrl+B** - Bold
- **Ctrl+I** - Italic
- **Ctrl+U** - Underline
- **Ctrl+Z** - Undo (browser default)
- **Ctrl+Y** - Redo (browser default)

### Multiple Documents
- You can open multiple editor windows
- Each has its own content
- Each can be saved separately

### File Indicator
- Watch the **●** symbol next to filename
- Appears when document is modified
- Disappears after saving

### Formatting Tips
1. Select text first, then apply formatting
2. Click formatting button again to remove
3. Use Clear Formatting (✕) to reset all styles
4. Headings automatically reset to normal text

## ❓ Common Questions

**Q: Where do saved files go?**
A: Files download to your browser's default Downloads folder.

**Q: Can I edit existing files?**
A: Yes! Right-click any text file in Explorer and select "Edit".

**Q: Can I have multiple editors open?**
A: Yes! Each file opens in its own window.

**Q: Does it auto-save?**
A: No, you must manually save or export your work.

**Q: Can I change text color?**
A: Not currently, but formatting like bold, italic, and size are supported.

**Q: What happens if I close without saving?**
A: Your work will be lost. Always save before closing!

**Q: Can I print my document?**
A: Export as HTML and open in a browser to print.

**Q: Does it support images?**
A: You can insert images via URL using the image button.

**Q: Can I collaborate with others?**
A: Not currently - this is a local editor only.

## 🆘 Troubleshooting

**Problem: Can't see the editor**
- Check if it's minimized in taskbar
- Try clicking the desktop icon again
- Look for it in Task Manager app

**Problem: Formatting not working**
- Make sure text is selected first
- Try clicking the button again
- Use Clear Formatting to reset

**Problem: Save dialog not appearing**
- Make sure you clicked the 💾 Save button
- Check browser popup blockers
- Try clicking Save again

**Problem: File not downloading**
- Check browser's download settings
- Look in your Downloads folder
- Try Export TXT or Export HTML instead

**Problem: Lost my work**
- Unfortunately, if you didn't save, work is lost
- Remember to save frequently!
- Use Ctrl+S habit from other apps (though it triggers browser save)
