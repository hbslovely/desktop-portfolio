About Me

This is a Windows-style desktop environment built with Angular. You can:

  • Double-click desktop icons to open applications
  • Drag windows around the desktop
  • Resize windows by dragging the edges and corners
  • Click taskbar buttons to switch between windows
  • Use the start menu to launch applications

Features

  • ✅ Desktop icons with drag & drop
  • ✅ Resizable and moveable windows
  • ✅ Taskbar with window management
  • ✅ Start menu
  • ✅ Calculator application
  • ✅ Window focus management
